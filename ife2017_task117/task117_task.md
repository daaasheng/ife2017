# 实现树形组件 #

● 练习综合运用HTML、CSS排版

● 练习 JavaSript 递归使用

● 练习 css 控制 html 隐藏和显示

## 任务描述 ##

● 类似window 资源管理器

● 支持无限层级展示

● 支持文件夹展开和隐藏

● 不必支持显示文件夹图片

● 支持数据格式如下：
var nodes = [ {name: "父节点1", children: [ {name: "子节点1"}, {name: "子节点2"} ]}, {name: "父节点2", children: [ {name: "子节点3"}, {name: "子节点4", children:[ {name:"子节点5"} ]} ]} ];
